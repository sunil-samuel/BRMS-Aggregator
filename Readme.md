# Overview

Just a sample application using Drools (BRMS) and kie-ci.  The objective is to show the following:
1. BPMN Process with Exclusive (XOR) Gateway using Rule (Drools) as Condition
2. Different Paths in Process Re-using the Same Rule Group
3. Include Rules Defined within Another Project (kjar)
